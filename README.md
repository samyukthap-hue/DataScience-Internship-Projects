# Data Science Internship Projects

## Overview
This repository contains multiple Data Science internship projects covering data preprocessing, exploratory analysis, machine learning, visualization, and dashboard creation.

The projects were developed as part of internship and academic work to demonstrate practical implementation of data science concepts.

## Projects Included

### 1. Titanic Passenger Survival Study
Objective:
Predict passenger survival using machine learning techniques.

Tasks Performed:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Prediction Model Building
- Visualization
- Performance Evaluation

Technologies:
- Python
- Pandas
- Seaborn
- Scikit-learn

---

### 2. Iris Species Classification

Objective:
Classify iris flowers into different species using supervised machine learning.

Tasks Performed:
- Data preprocessing
- Visualization
- Classification model
- Model evaluation

Technologies:
- Python
- Scikit-learn
- Matplotlib

---

### 3. Exploratory Data Analysis (EDA)

Objective:
Perform complete data analysis and extract meaningful insights.

Tasks Performed:
- Data Cleaning
- Missing value handling
- Correlation analysis
- Visualization
- Insight generation

---

### 4. Smart Retail Intelligence System

Objective:
Develop an advanced retail analytics system using machine learning.

Features:
- Customer Segmentation
- Sales Trend Analysis
- Profit Prediction
- Sales Forecasting
- Dashboard Creation

This project is designed as an innovative academic project.
